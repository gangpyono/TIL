# 한번에 둘 이상 연결해야 하면?

- 한 IP에서 여러 프로그램을 실행시킬떄 패킷간 구분을 어떻게 지을 것인가? -> PORT



### PORT
- 같은 IP 내에서 프로세스를 구분해줌.
- 0 ~ 65535 : 할당 가능.
- 0 ~ 1023 : 잘 알려진 포트, 사용하지 않는 것이 좋음.
  - FTP : 20,21
  - TELNET : 23
  - HTTP : 80
  - HTTPS : 443

# IP로 통신했을시 문제점
- IP를 일일이 기억하기가 어렵다.
- IP가 변경될 수도 있다.


### DNS(Domain Name System)
- 전화번호부 생각 : 도메인명 - IP
  - 이후 서버에서 IP를 수정하여도 연결된 도메인명으로 찾을 수 있게됨.
