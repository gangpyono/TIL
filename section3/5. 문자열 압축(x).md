# 문자열 압축
- 알파벳 대문자로 이루어진 문자열을 입력받아 같은 문자가 연속으로 반복되는 경우 반복되는 문자 바로 오른쪽에 반복 횟수를 표기하는 방법으로 문자열을 압축하는 프로그램을 작성하시오. 
- 단 반복횟수가 1인 경우 생략합니다.


# 풀이
```html
   <script>
            function solution(s){
                let answer="";
                let cnt = 1;
                
                for(let x = 0; x<s.length; x++) {
                    if(s[x] === s[x+1]) cnt++;
                    else {
                        answer += s[x];
                        if(cnt>1) answer+=String(cnt);
                        cnt = 1;
                    }
                    
                }
            return answer;
        }
            let str="KKHSSSSSSSE";
            console.log(solution(str));
        </script>
```


입력 : KKHSSSSSSSE


출력 : K2HS7E
