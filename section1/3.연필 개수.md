# 1. 연필개수 구하기
- 연필 1 다스는 12자루다. 학생 1인당 연필을 1자루씩 나누어 준다고 할 때 N명이 학생수 를 입력하면 필요한 연필의 다스 수를 계산하는 코드를작성하시오.


## 풀이
```js
       <script>
            function solution(n){
                let answer;
                let remain = n%12;

                answer = Math.floor(n/12);
                if(remain != 0) ++answer;    
                
                return answer;
            }

            console.log(solution(178));
        </script>
```

## 해설
```js
  <script>
            function solution(n){
                let answer = Math.ceil(n/12);
                return answer;
            }
            console.log(solution(25));
        </script>
```
