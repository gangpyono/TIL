# 1. 배열 arr의 값중 최솟값을 출력하시오.


## 풀이.
```js
<html>
    <head>
        <meta charset="UTF-8">
        <title>출력결과</title>
    </head>
    <body>
        <script>
            function solution(arr){         
                let answer;
               let min;
                min = arr[0];
                for(let i= 1; i< arr.length; i++) {
                    if(min > arr[i]) min = arr[i];
                }

                answer = min;

                return answer;

            }

            let arr = [5, 3, 7, 11, 2, 15, 17];
            console.log(solution(arr));
        </script>
    </body>
</html>
```


## 해설
```js
<html>
    <head>
        <meta charset="UTF-8">
        <title>출력결과</title>
    </head>
    <body>
        <script>
            function solution(arr){         
                let answer = Math.min(...arr);  /// ...은 전개를 의미  (arr[0].arr[1],arr[2],arr[3],arr[4]...,arr[6]);                
                return answer;
            }

            let arr = [5, 3, 7, 11, 2, 15, 17];
            console.log(solution(arr));
        </script>
    </body>
</html>
```
