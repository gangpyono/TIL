# 1. Render Tree.

- Render Tree는 사용자에게 보여줄 요소로 구성된다.
- DOM트리구성요소중 head는 사용자에게 직접적으로 보이지않는 부분이기때문에 Render Tree에서 제외된다.
- css속성중 display : none으로 설정시에도 마찬가지로 사용자에게 보이지않는부분으로 제외.
