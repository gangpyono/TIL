# 함수형 프로그래밍 - 부수 효과를 없애고 조합성을 강조하는 프로그래밍. - 부수 효과를 최소화 -> 순수함수를 만든다. - 조합성을 강조 -> 모듈화 수준을 높인다.(재사용성을 높인다) - 순수함수들의 조합으로 프로그래밍 하는것. # 순수함수 - 부수효과가 없는 함수.(부수효과 : 리턴값외에 함수 외부의 상태에 변화를 일으키는것) - 전달받은 인자가 동일하면 언제호출하든 항상 같은 값을 리턴하는 함수. - 함수가 받은 인자외에 다른 외부값의 변화를 일으키지 않는 함수. # 자바스크립트에서 함수는 일급 함수다. - 일급 - 값으로 다룰 수 있다. - 변수에 담을 수 있다. - 함수의 인자로 사용될 수 있다. - 함수의 결과로 사용될 수 있다. # 평가 - 코드가 계산(Evaluation) 되어 값을 만드는 것 # 일급 함수 - 함수를 값으로 다룰 수 있다. - 조합성과 추상화의 도구
